Solución completa de gestión académica con .NET 9, Entity Framework, MySQL y Docker

Resumen del Proyecto

Sistema completo de gestión de estudiantes que cumple con todos los requisitos del simulacro, diseñado con una arquitectura DDD simple para principiantes.

Aplicación Web

Panel administrativo completo con CRUD de estudiantes, importación de Excel, generación de PDFs y dashboard con IA.

API REST

Servicios para estudiantes con autoregistro, autenticación JWT y consulta de información personal.

Inteligencia Artificial

Dashboard con consultas en lenguaje natural para análisis de datos académicos.

Inicio Rápido

Ejecuta toda la aplicación con un solo comando

Con Docker Compose
docker-compose up -d

Esto iniciará MySQL, la API REST y la Aplicación Web

Acceso a las Aplicaciones
  • Web Admin: http://localhost:5001
  • API REST: http://localhost:5000
  • Swagger UI: http://localhost:5000
  • MySQL: localhost:3306

Documentación de la API

Endpoints principales para integración con el portal de estudiantes

Autoregistro
POST /api/auth/registro { "numeroDocumento": "123456789", "nombre": "Juan", "apellido": "Pérez", "email": "juan@email.com", "password": "Pass123!", "programaAcademicoId": 1, "tipoDocumentoId": 1, "modalidadId": 1 }
Login
POST /api/auth/login { "email": "juan@email.com", "password": "Pass123!" }
Perfil (JWT)
GET /api/auth/perfil Authorization: Bearer {token}
